# Employee Dashboard

## Overview

The Employee Dashboard is the personal portal for all regular employees. It provides quick access to relevant features, personal information, and daily tasks without administrative privileges.

## Purpose

The Employee Dashboard serves as the central hub for employees to:
- View personal information and statistics
- Access attendance and leave features
- View assigned tasks and projects
- Check calendar and upcoming events
- Submit requests and forms
- View announcements and updates
- Manage personal profile

## What Users Can See

### Dashboard Sections

#### 1. **Welcome Banner**
- Personalized greeting with employee name
- Current date and time
- Quick status overview

#### 2. **Personal Statistics Cards**
Quick view of key metrics:
- **Attendance Summary**: Your attendance this month
- **Leave Balance**: Remaining leave days
- **Pending Requests**: Requests awaiting approval

- **Upcoming Events**: Events you're attending

#### 3. **Today's Overview**
- **Attendance Status**: Check-in/check-out status for today
- **Today's Tasks**: Tasks due today
- **Today's Events**: Events scheduled for today
- **Today's Schedule**: Your daily agenda

#### 4. **Recent Activity Feed**
Shows your recent activities:
- Leave requests submitted
- Tasks completed
- Events attended
- Requests submitted
- Announcements read

#### 5. **Quick Actions Panel**
Convenient buttons for common tasks:
- Check In/Check Out (Attendance)
- Apply for Leave
- Submit Request
- View Calendar
- Create Task (if applicable)
- View Profile

#### 6. **Announcements Widget**
- Latest company announcements
- Important notices
- Team updates
- Read/unread indicators

#### 7. **Calendar Widget**
Mini calendar showing:
- Your leave days
- Events you're attending
- Task deadlines
- Important dates

## How to Use the Employee Dashboard

### Daily Workflow

#### Starting Your Day

1. **Login** to the portal
2. Dashboard loads automatically
3. **Check announcements** for important updates
4. **Review today's overview**:
   - Check scheduled events
   - Review assigned tasks
   - Note any deadlines
5. **Check In** for attendance
6. Begin your work

#### During the Day

- **Update task progress** as you work
- **Check calendar** for meetings or events
- **Respond to notifications**
- **Submit requests** as needed
- **Review announcements** periodically

#### Ending Your Day

1. **Update task status** before leaving
2. **Check Out** for attendance
3. **Review tomorrow's schedule**
4. **Complete any pending actions**
5. **Logout** from the system

### Managing Your Profile

1. Click **"My Profile"** from the menu
2. View your current information:
   - Personal details
   - Job information
   - Contact information
   - Emergency contacts
3. Click **"Edit"** to update information
4. Make necessary changes
5. Click **"Save"** to update
6. Profile is updated system-wide

### Recording Attendance

#### Check In
1. Navigate to **"Attendance"**
2. Click **"Check In"** button
3. System records time and location
4. Confirmation message appears
5. Status shows "Checked In"

#### Check Out
1. At end of day, go to **"Attendance"**
2. Click **"Check Out"** button
3. System calculates total hours
4. Confirmation displayed
5. Day's attendance is complete

### Applying for Leave

1. Click **"Leave"** from menu
2. Select **"Apply Leave"**
3. Fill in leave request form:
   - Leave type (Annual, Sick, etc.)
   - Start date
   - End date
   - Number of days (auto-calculated)
   - Reason for leave
   - Supporting documents (if required)
4. **Review** your request
5. Click **"Submit"**
6. Request sent to supervisor/admin
7. Track status in **"My Leave"**
8. Receive notification when approved/rejected

### Managing Tasks

#### Viewing Tasks
1. Navigate to **"Tasks"**
2. See list of assigned tasks:
   - Task name
   - Description
   - Due date
   - Priority
   - Status
3. Click on task for details

#### Updating Task Progress
1. Open the task
2. Click **"Update Status"**
3. Select new status:
   - Not Started
   - In Progress
   - Completed
   - On Hold
4. Add **progress notes**
5. Update **completion percentage** (if applicable)
6. Click **"Save"**
7. Supervisor is notified

### Submitting Requests

1. Go to **"Requests"** section
2. Click **"Submit Request"**
3. Select **request type**:
   - Equipment request
   - IT support
   - Training request
   - Travel request
   - Other requests
4. Fill in request form:
   - Title
   - Description
   - Urgency
   - Supporting documents
5. **Review** details
6. Click **"Submit"**
7. Track in **"My Requests"**
8. Receive approval notification

### Viewing Calendar

1. Click **"Calendar"** from menu
2. View options:
   - Month view
   - Week view
   - Day view
3. See color-coded items:
   - Leave days
   - Events
   - Task deadlines
   - Holidays
4. Click on any item for details
5. Use navigation to move between dates

### Working with Forms

#### Submitting Work Handover Form

1. Navigate to **"Forms"**
2. Select **"Work Handover Form"**
3. Fill in details:
   - Handover date
   - Tasks being handed over
   - Current status of each task
   - Important notes
   - Contact information
4. Add **attachments** if needed
5. Select **approver**
6. **Review** thoroughly
7. Click **"Submit"**
8. Track approval status

## Dashboard Customization

Personalize your dashboard experience:
- **Widget Preferences**: Show/hide widgets
- **Default Landing Page**: Set your preferred start page
- **Notification Settings**: Choose what alerts you receive
- **Theme**: Light or dark mode (if available)
- **Language**: Select preferred language

## Notifications

You can receive notifications for:
- **Leave Approval/Rejection**: When leave is processed
- **Task Assignment**: New tasks assigned to you
- **Task Deadline**: Upcoming due dates
- **Event Invitations**: New event invites
- **Request Updates**: Status changes on requests
- **Announcements**: Important company updates
- **Form Approvals**: Status of submitted forms

### Managing Notifications

1. Click **notification icon** (bell icon)
2. View all notifications
3. Click to read details
4. Mark as read
5. Configure preferences in **Settings**

## Troubleshooting

### Dashboard Not Loading
- Refresh the page (F5)
- Clear browser cache
- Check internet connection
- Try different browser
- Contact IT support

### Cannot Submit Request
- Check all required fields are filled
- Ensure file attachments are within size limit
- Verify you have submission permissions
- Contact supervisor if issue persists

### Notifications Not Showing
- Check notification settings
- Enable browser notifications
- Verify email notification settings
- Update notification preferences
