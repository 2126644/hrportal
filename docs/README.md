### Understanding the Structure
Each documentation page follows a consistent structure:
- **Overview**: What the feature is about
- **Purpose**: Why you would use this feature
- **What Users Can See**: Different views for different roles
- **How to Use**: Step-by-step instructions
- **Features**: Detailed feature descriptions
- **Best Practices**: Recommended approaches
- **Troubleshooting**: Common issues and solutions
- **Tips**: Helpful hints for effective use
