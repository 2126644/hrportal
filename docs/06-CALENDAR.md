# Calendar

## Overview

The Calendar feature provides a unified view of all scheduled activities including events, leaves, task deadlines, and important dates. It helps employees and administrators plan and coordinate activities effectively.

## Purpose

The Calendar system is designed to:
- Display all scheduled activities in one place
- Provide different calendar views (daily, weekly, monthly)
- Show employee leaves and availability
- Display company events and meetings
- Track task and project deadlines
- Highlight public holidays and company closures
- Enable better planning and coordination

## What Users Can See

### For Employees

#### Personal Calendar View
- **My Schedule**: Your personal calendar with:
  - Your approved leaves
  - Events you're invited to
  - Task deadlines
  - Project milestones
  - Company holidays
  - Personal appointments (if enabled)

#### Calendar Features
- **Color Coding**: Different colors for different types
  - Blue: Events
  - Green: Available days
  - Red: Leave days
  - Yellow: Task deadlines
  - Gray: Holidays
- **View Options**: Day, Week, Month views
- **Navigation**: Move between dates easily
- **Click for Details**: View full information on any item

### For Administrators

#### Organization Calendar
- **Company-wide View**: See all organizational activities
- **Team Calendars**: View by department or team
- **Leave Overview**: Who is on leave when
- **Event Schedule**: All scheduled events
- **Resource Planning**: Availability visualization

#### Admin Calendar Features
- All employee calendar features, plus:
- **Filter Options**: By employee, department, event type
- **Multi-calendar View**: Multiple calendars side-by-side
- **Conflict Detection**: Overlapping schedules highlighted
- **Capacity Planning**: Team availability metrics
- **Export Options**: Download calendar data

## How to Use the Calendar

### For Employees: Viewing Your Calendar

#### Accessing the Calendar

1. **Login** to your dashboard
2. Click **"Calendar"** from the main menu
3. Your personal calendar loads in month view (default)

#### Navigating the Calendar

**Change Views:**
- Click **"Month"** for monthly overview
- Click **"Week"** for detailed week view
- Click **"Day"** for single-day agenda
- Click **"Agenda"** for list view of upcoming items

**Move Between Dates:**
- Click **left/right arrows** to navigate
- Click **"Today"** to return to current date
- Click on specific date to jump to it
- Use **date picker** for quick navigation

**View Details:**
- Click on any calendar item
- Popup shows full details:
  - Title and description
  - Date and time
  - Location (for events)
  - Attendees/participants
  - Additional notes
- Click **"Close"** or anywhere outside to dismiss

#### Understanding Calendar Items

**Leave Days** (appears when you have approved leave)
- Shows as colored block on date
- Hover to see leave type
- Click for full leave details
- View approved leave only

**Events** (shows events you're invited to)
- Event title appears on date
- Time displayed if set
- Location shown in details
- RSVP status indicated

**Task Deadlines** (tasks due on specific dates)
- Task name appears
- Priority indicated by color intensity
- Completion status shown
- Click to view/update task

**Company Holidays** (public holidays and closures)
- Entire day highlighted
- Holiday name displayed
- Non-working day indicator

#### Syncing with Personal Calendar

If calendar sync is enabled:
1. Go to **"Calendar Settings"**
2. Click **"Export Calendar"** or **"Subscribe"**
3. Choose format:
   - **iCal Format**: For Apple Calendar
   - **Google Calendar**: Direct sync
   - **Outlook**: Export file
4. Follow integration instructions
5. Your HR calendar appears in personal calendar app

### For Administrators: Managing Calendars

#### Viewing Organization Calendar

1. **Access Admin Calendar**
   - Login to Admin Dashboard
   - Click **"Calendar"** from menu
   - Organization-wide view loads

2. **Apply Filters**
   - **By Department**: View specific team calendars
   - **By Employee**: See individual schedules
   - **By Type**: Filter events, leaves, tasks
   - **By Status**: Confirmed, pending, cancelled

3. **Analyze Availability**
   - See who is available/unavailable
   - Identify overlapping absences
   - Plan team activities
   - Spot scheduling conflicts

#### Managing Calendar Events

**Creating Events** (detailed in Events section)
- Click on date or click **"Create Event"**
- Events appear automatically on calendar
- Update calendar when events are modified

**Managing Leaves**
- Approved leaves show on calendar automatically
- Color-coded by leave type
- Click to view leave details
- Option to cancel approved leave (with permission)

**Task Deadlines**
- Tasks with due dates appear automatically
- Updated when task dates change
- Removed when tasks complete
- Filter tasks by project or person

#### Calendar Views for Planning

**Month View**
- Best for overview and planning
- See entire month at a glance
- Identify busy periods
- Spot staffing gaps

**Week View**
- Detailed weekly schedule
- Hour-by-hour if time-based events
- Good for short-term planning
- See team availability clearly

**Day View**
- Complete agenda for specific day
- Chronological listing
- All details visible
- Good for daily planning

**Agenda/List View**
- Upcoming events in list format
- Sort by date, type, or priority
- Quick scanning of schedule
- Export-friendly format

#### Calendar Reports

1. Navigate to **"Calendar Reports"**
2. Select report type:
   - **Availability Report**: Team availability over period
   - **Leave Calendar**: Leave schedule
   - **Event Summary**: Upcoming events list
   - **Utilization Report**: Resource usage
3. Set parameters:
   - Date range
   - Departments/employees
   - Activity types
4. **Generate** and **Export**

## Calendar Features

### Automatic Updates

Calendar automatically reflects:
- **Leave approvals**: Approved leaves added instantly
- **Event changes**: Updates when events modified
- **Task updates**: Deadline changes reflected
- **Cancellations**: Removed items disappear
- **New invitations**: Appear when you're invited

### Color Coding System

Standard color scheme:
- **Blue**: Company events and meetings
- **Green**: Available/working days
- **Red/Orange**: Leave days
- **Yellow**: Task deadlines
- **Purple**: Project milestones
- **Gray**: Holidays and non-working days
- **Custom**: Configurable by admin

### Recurring Events

For repeating events:
- Daily, weekly, monthly, yearly patterns
- Show all occurrences on calendar
- Edit single instance or entire series
- Delete one or all occurrences
- Exceptions handled (e.g., skip holidays)

### Calendar Notifications

Receive reminders for:
- **Upcoming events**: 1 day, 1 hour before
- **Task deadlines**: When approaching
- **Leave start**: Day before leave begins
- **Event changes**: When modified or cancelled
- **New invitations**: When invited to events

Configure in **Settings** → **Notifications**

### Multi-Calendar View

View multiple calendars simultaneously:
1. **Personal Calendar**: Your schedule
2. **Team Calendar**: Your department
3. **Company Calendar**: Organization-wide
4. **Public Holidays**: National holidays

Toggle each on/off to customize view

**For Administrators:**
- Publish department calendars

- Control visibility levels
- Manage calendar permissions

## Troubleshooting

### Common Issues

**Calendar Not Loading**
- Refresh the page (F5)
- Clear browser cache
- Check internet connection
- Try different browser
- Contact IT support

**Events Not Showing**
- Check filter settings (may be filtered out)
- Verify you're invited to the event
- Ensure event is confirmed, not cancelled
- Check date range being viewed
- Refresh calendar

**Cannot Create Event**
- Verify you have creation permissions
- Check all required fields filled
- Ensure date/time is valid
- Try different browser
- Contact administrator

**Sync Issues with External Calendar**
- Re-authenticate sync connection
- Check sync settings
- Verify calendar subscription URL
- Remove and re-add sync
- Contact IT support

**Wrong Time Zone**
- Check your profile time zone setting
- Update time zone in settings
- Refresh calendar after change
- Clear browser cache

**Notifications Not Working**
- Enable notifications in Settings
- Allow browser notifications
- Check spam folder for email notifications
- Verify notification preferences
- Test with a sample event
