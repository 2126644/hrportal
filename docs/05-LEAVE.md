# Leave Management

## Overview

The Leave Management system allows employees to request time off and enables administrators to review, approve, or reject leave requests. It tracks leave balances and maintains a comprehensive leave history.

## Purpose

The Leave Management system is designed to:
- Streamline the leave request and approval process
- Track employee leave balances accurately
- Maintain a calendar of approved leaves
- Ensure adequate staffing through leave visibility
- Generate leave reports for planning and compliance
- Automate leave calculations and notifications

## What Users Can See

### For Employees

#### Leave Dashboard
- **Leave Balance Summary**: Available leave days by type
- **Apply Leave Button**: Quick access to request form
- **My Leave Requests**: History of all leave requests
- **Leave Calendar**: Visual view of your leaves
- **Leave Statistics**: Leave taken vs. available

#### Leave Types
Common leave categories:
- **Annual Leave**: Vacation/planned time off
- **Sick Leave**: Medical-related absences
- **Personal Leave**: Personal matters
- **Maternity/Paternity Leave**: Parental leave
- **Compassionate Leave**: Family emergencies
- **Unpaid Leave**: Leave without pay
- **Other**: Special circumstances

### For Administrators

#### Admin Leave Dashboard
- **All Leave Requests**: Complete list of requests
- **Pending Approvals**: Requests awaiting decision
- **Approved Leaves**: Confirmed time off
- **Leave Calendar**: Organization-wide leave view
- **Leave Reports**: Statistics and analytics
- **Leave Balance Management**: Track all employee balances

#### Admin View Features
- Filter by employee, department, date range
- Bulk approval/rejection options
- Leave conflict detection
- Team availability overview
- Export leave data

## How to Use Leave Management

### For Employees: Requesting Leave

#### Applying for Leave

1. **Navigate to Leave Section**
   - Login to your dashboard
   - Click **"Leave"** from the main menu
   - Click **"Apply Leave"** button

2. **Fill Leave Request Form**
   - **Leave Type**: Select from dropdown (Annual, Sick, etc.)
   - **Start Date**: Choose first day of leave
   - **End Date**: Choose last day of leave
   - **Number of Days**: Auto-calculated based on dates
   - **Reason**: Provide brief explanation
   - **Contact During Leave**: Phone/email if reachable
   - **Attachments**: Upload supporting documents (if required)
     - Medical certificates for sick leave
     - Other supporting documentation

3. **Review Your Request**
   - Verify all information is correct
   - Check leave balance is sufficient
   - Ensure dates don't conflict with critical work

4. **Submit Request**
   - Click **"Submit"** button
   - Confirmation message appears
   - Request sent to supervisor/approver
   - Email notification sent

5. **Track Status**
   - View in **"My Leave Requests"**
   - Status indicators:
     - **Pending**: Awaiting approval
     - **Approved**: Confirmed by supervisor
     - **Rejected**: Declined with reason
     - **Cancelled**: Withdrawn by employee

#### Checking Leave Balance

1. Navigate to **"Leave"** section
2. View **"Leave Balance"** widget
3. See breakdown by leave type:
   - Total allocated
   - Used this year
   - Remaining balance
   - Pending requests (deducted from balance)
4. Check **effective date** of balance calculation

#### Viewing Leave History

1. Go to **"My Leave Requests"**
2. See complete list with:
   - Request date
   - Leave type
   - Duration
   - Status
   - Approver
   - Comments
3. Use filters:
   - Date range
   - Leave type
   - Status
4. Click on any request for full details

#### Cancelling Leave Request

For pending requests:
1. Go to **"My Leave Requests"**
2. Find the request to cancel
3. Click **"Cancel"** button
4. Confirm cancellation
5. Request is withdrawn
6. Leave balance is restored
7. Approver is notified

For approved requests:
1. Contact your supervisor directly
2. Request cancellation approval
3. Admin can cancel approved leave if agreed

### For Administrators: Managing Leave

#### Reviewing Leave Requests

1. **Access Leave Management**
   - Login to Admin Dashboard
   - Click **"Leave"** from menu
   - See **"Pending Approvals"** section

2. **View Request Details**
   - Click on leave request
   - Review information:
     - Employee name and department
     - Leave type and duration
     - Reason for leave
     - Supporting documents
     - Current leave balance
     - Team calendar impact

3. **Check for Conflicts**
   - System highlights overlapping leaves
   - Review team availability
   - Consider workload and deadlines
   - Check coverage arrangements

4. **Approve or Reject**
   
   **To Approve:**
   - Click **"Approve"** button
   - Add approval comments (optional)
   - Click **"Confirm"**
   - Employee receives approval notification
   - Leave is added to calendar
   - Leave balance is updated

   **To Reject:**
   - Click **"Reject"** button
   - **Add rejection reason** (required)
   - Click **"Confirm"**
   - Employee receives rejection notification
   - Leave balance remains unchanged

#### Managing Leave Balances

1. Navigate to **"Leave Management"** → **"Leave Balances"**
2. View all employees' leave balances
3. **Adjust balances** if needed:
   - Click on employee
   - Select **"Adjust Balance"**
   - Enter adjustment:
     - Leave type
     - Number of days (add or subtract)
     - Reason for adjustment
   - Click **"Save"**
   - Employee is notified of change

#### Viewing Leave Calendar

1. Click **"Leave Calendar"** tab
2. See visual calendar with:
   - All approved leaves
   - Color-coded by leave type or employee
   - Department filter options
3. **Navigate** between months
4. **Click** on any date to see who is on leave
5. **Export** calendar view

#### Generating Leave Reports

1. Go to **"Leave Reports"** section
2. Select report type:
   - **Employee Leave Summary**: Individual leave usage
   - **Department Leave Report**: Team-wise breakdown
   - **Leave Type Analysis**: Usage by leave category
   - **Leave Trends**: Patterns over time
   - **Leave Balance Report**: Current balances
3. Set parameters:
   - Date range
   - Employees/departments
   - Leave types
4. Click **"Generate"**
5. View report on screen
6. **Export** to PDF or Excel

#### Setting Leave Policies

1. Navigate to **"Settings"** → **"Leave Settings"**
2. Configure:
   - **Leave Types**: Add/edit leave categories
   - **Annual Allocation**: Days per leave type
   - **Accrual Rules**: How leave accumulates
   - **Carry Forward**: Rules for unused leave
   - **Maximum Balance**: Upper limits
   - **Notice Period**: Advance notice required
   - **Approval Workflow**: Who approves what
3. **Save** policy changes
4. Changes apply to all employees

## Leave Features

- **Days Calculation**: Auto-counts working days
- **Weekend Exclusion**: Weekends not counted (configurable)
- **Public Holiday Exclusion**: Holidays excluded
- **Half-day Options**: Request partial days
- **Balance Updates**: Real-time balance adjustment

### Leave Workflows

#### Standard Approval Flow
1. Employee submits request
2. Supervisor receives notification
3. Supervisor approves/rejects
4. Employee receives decision
5. Calendar and balance update automatically

#### Multi-level Approval (if configured)
1. Employee submits
2. Direct supervisor approves
3. Department head approves
4. HR approves (if needed)

  - Personal calendar

## Leave Policies

### Common Policy Rules

**Annual Leave**
- Allocated at year start or pro-rated
- Requires advance notice (e.g., 2 weeks)
- Maximum consecutive days may apply
- Carry forward rules (if any)
- Use-it-or-lose-it deadlines

**Sick Leave**
- May require medical certificate
- Short notice acceptable
- Limited to allocated days
- May need doctor's note for >3 days
- Cannot be carried forward

**Emergency Leave**
- Minimal notice required
- Requires explanation
- Limited days per year
- May need supporting evidence

### Leave Restrictions

- **Blackout Periods**: Busy seasons when leave is restricted
- **Minimum Team Strength**: Required staffing levels
- **Maximum Consecutive Days**: Limits on extended absence
- **Notice Requirements**: Advance warning needed
- **Approval Levels**: Different approvers for different durations

## Troubleshooting

### Common Issues

**"Insufficient Leave Balance" Error**
- Check your current balance
- Verify leave type selected
- Consider pending requests (already deducted)
- Contact HR if balance seems incorrect

**Request Not Approved/Rejected**
- Check with your supervisor
- Verify they received notification
- Request may be in their pending queue
- Contact HR if delayed beyond normal timeframe

**Cannot Submit Leave Request**
- Ensure all required fields are filled
- Check date range is valid
- Verify you have leave balance
- Check for system blackout periods
- Try different browser if technical issue

**Leave Not Showing in Calendar**
- Only approved leaves appear in calendar
- Check leave status (must be "Approved")
- Refresh calendar view
- Clear browser cache
- Contact IT support

**Medical Certificate Upload Failed**
- Check file size (usually max 5-10MB)
- Ensure file format is supported (PDF, JPG, PNG)
- Try compressing the file
- Contact IT if problem persists

- **Documentation**: Save approval emails
- **Balance monitoring**: Regular checks prevent surprises
- **Policy awareness**: Know your entitlements
- **Professional courtesy**: Give adequate notice
- **Handover planning**: Arrange work coverage
