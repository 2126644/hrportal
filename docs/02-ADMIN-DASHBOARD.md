# Admin Dashboard

## Overview

The Admin Dashboard is the central control panel for system administrators. It provides a comprehensive overview of the entire HR Portal and quick access to all administrative functions.

## Purpose

The Admin Dashboard serves as the command center for administrators to:
- Monitor overall system activity and statistics
- Access all administrative tools and features
- View key metrics and reports at a glance
- Manage employees, attendance, leave, and requests
- Configure system settings

## What Users Can See

### Dashboard Sections

#### 1. **Overview Statistics**
Quick metrics displayed at the top of the dashboard:
- Total number of employees
- Active employees count
- Pending leave requests
- Pending other requests
- Today's attendance summary
- Upcoming events count
- Active projects and tasks

#### 2. **Recent Activity**
Shows recent system activities including:
- New employee registrations
- Recent leave requests
- Latest attendance records
- Recent request submissions
- New announcements

#### 3. **Quick Actions Panel**
Convenient shortcuts to common admin tasks:
- Create new employee
- Post announcement
- Create event
- Create project
- Create task
- View reports

#### 4. **Calendar Widget**
Displays:
- Upcoming events
- Leave schedule
- Important dates
- Meetings and appointments

#### 5. **Pending Approvals**
Highlights items requiring admin attention:
- Leave requests awaiting approval
- Request forms pending review
- Employee registrations needing verification

## How to Use the Admin Dashboard

### Accessing the Dashboard

1. **Login** with admin credentials
2. After successful authentication, you'll be automatically directed to the Admin Dashboard
3. The dashboard displays immediately with all widgets and statistics

### Navigating the Dashboard

1. **Top Navigation Bar**: Access main sections (Dashboard, Employees, Attendance, etc.)
2. **Sidebar Menu**: Additional navigation options
3. **Quick Action Buttons**: Fast access to common tasks
4. **Search Function**: Find employees, requests, or records quickly

### Viewing Statistics

1. **Overview Cards**: Click any statistic card to view detailed information
2. **Graphs and Charts**: Hover over data points for specific values
3. **Date Filters**: Adjust timeframes for reports and statistics
4. **Export Options**: Download reports in PDF or Excel format

### Managing Employees

From the dashboard:
1. Click **"Employees"** in the main menu
2. Use the **"Add Employee"** button to create new accounts
3. Click on an employee name to view/edit their profile
4. Use filters to find specific employees
5. Bulk actions available for multiple employees

### Handling Approvals

1. **Pending Approvals Widget**: Shows count of items needing attention
2. Click on **"Leave Requests"** or **"Requests"** to review
3. View request details and supporting documents
4. Click **"Approve"** or **"Reject"** with optional comments
5. Employee receives notification of decision

### Creating Announcements

1. Click **"Announcements"** from the menu
2. Click **"Create Announcement"** button
3. Fill in:
   - Title
   - Content/message
   - Target audience (all employees or specific groups)
   - Priority level
   - Publish date
4. Click **"Publish"** to post the announcement

### Scheduling Events

1. Navigate to **"Events"** section
2. Click **"Create Event"**
3. Enter event details:
   - Event name
   - Date and time
   - Location
   - Description
   - Attendees
4. **Save** the event
5. Invitations sent automatically to attendees

## Dashboard Customization

Administrators can customize their dashboard view:
- **Widget Arrangement**: Drag and drop widgets to preferred positions
- **Show/Hide Widgets**: Toggle visibility of different sections
- **Date Range Selection**: Customize report timeframes
- **Default Views**: Set preferred landing pages

## Reports and Analytics

Access comprehensive reports:
- **Attendance Reports**: Daily, weekly, monthly summaries
- **Leave Reports**: Leave trends and balances
- **Employee Reports**: Headcount, demographics, turnover
- **Project Reports**: Progress and completion rates
- **Custom Reports**: Create specific data views

## Best Practices

- **Daily Dashboard Review**: Check dashboard each morning for pending items
- **Regular Updates**: Keep employee information current
- **Timely Approvals**: Process requests within expected timeframes
- **Data Backup**: Regularly export important reports
- **Security**: Log out when leaving your workstation
- **Announcements**: Use for important company-wide communications
- **Monitor Trends**: Review statistics to identify patterns

## Tips for Efficiency

- Use **Quick Actions** for frequently performed tasks
- Set up **keyboard shortcuts** for common operations
- Utilize **search functionality** instead of manual browsing
- **Filter and sort** data tables for faster access
- **Bookmark** frequently accessed pages
- Enable **email notifications** for urgent approvals
