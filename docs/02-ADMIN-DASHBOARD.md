# Admin Dashboard

## Overview

The Admin Dashboard is the central control panel for system administrators. It provides a comprehensive overview of the entire HR Portal and quick access to all administrative functions.

## Purpose

The Admin Dashboard serves as the command center for administrators to:
- Quick glance to account info and overall system statuses
- Monitor employees attendance of the day
- View recent announcements and activities
- View pending requests at a glance

## What Users Can See

### Dashboard Sections

#### 1. **Profile Summary Card**
Quick profile summary at the top of the dashboard:
- Profile photo
- Name
- Role
- User ID
- Email address
- Account Settings button (Click Account Settings to update your account info.)

#### 2. **System Overview**
Shows current system statuses:
- Database status (Connected / Not connected)
- Active users (Total)
- Sync status (Last database update)

#### 3. **Today's Attendance**
Shows all employee's attendance of the day, who is present, absent or late with their details:
- Employee name
- Status
- Punch in time
- Punch out time
- Department

#### 4. **Announcements**
Displays:
- Announcement title
- Description
- Priority
- Post time

#### 5. **Leave & Time Slip Requests**
Highlights recent items requiring admin attention:
- Leave and time slip requests awaiting approval

#### 6. **Recent Activities**
Highlights recent activities done by employees in the system:
- Punched in/ punched out
- Applying for leave
- Applying for time slip

## How to Use the Admin Dashboard

### Accessing the Dashboard

1. **Login** with admin credentials
2. After successful authentication, you'll be automatically directed to the Admin Dashboard
3. The dashboard displays immediately

### Navigating the Dashboard

1. **Top Navigation Bar**: Access notifications, log out.
2. **Sidebar Menu**: Access main sections
    - Dashboard
    - Attendance
    - Leave
    - Calendar
    - Event
    - Task & Project
    - Form,
    - Request
    - Employee
    - Announcement
    - System Setting

## Best Practices

- **Daily Dashboard Review**: Check dashboard each morning for pending items
- **Regular Updates**: Keep employee information current
- **Timely Approvals**: Process requests within expected timeframes
- **Data Backup**: Regularly export important reports
- **Security**: Log out when leaving your workstation
