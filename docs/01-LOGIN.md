# Login & Authentication

## Overview

The Login page is the entry point to the HR Portal system. All users must authenticate themselves before accessing any functionality within the portal.

## Purpose

The login functionality ensures secure access to the HR Portal by:
- Verifying user credentials
- Maintaining session security
- Providing password recovery options
- Supporting two-factor authentication for enhanced security

## What Users Can See

### Login Page Features

1. **Email/Username Field**: Enter your registered email address
2. **Password Field**: Enter your password (hidden for security)
3. **Remember Me**: Optional checkbox to stay logged in
4. **Login Button**: Submit credentials to access the system
5. **Forgot Password Link**: Recover access if you've forgotten your password

### Two-Factor Authentication (Optional)

If two-factor authentication is enabled for your account:
- After entering credentials, you'll be prompted for a verification code
- Enter the code from your authenticator app or received via email/SMS
- Complete authentication to access the system

## How to Login

### Step-by-Step Guide

1. **Navigate to the Login Page**
   - Open your web browser
   - Go to your organization's HR Portal URL
   - You'll be automatically redirected to the login page if not authenticated

2. **Enter Your Credentials**
   - Type your email address in the "Email" field
   - Type your password in the "Password" field
   - Optionally, check "Remember Me" to stay logged in on this device

3. **Click the Login Button**
   - Click the "Login" or "Sign In" button
   - Wait for the system to verify your credentials

4. **Two-Factor Authentication (if enabled)**
   - Enter the verification code when prompted
   - Click "Verify" to complete authentication

5. **Access Your Dashboard**
   - Upon successful login, you'll be redirected to your dashboard
   - Admin users see the Admin Dashboard
   - Regular employees see the Employee Dashboard

## Password Recovery

If you've forgotten your password:

1. **Click "Forgot Password"** on the login page
2. **Enter Your Email**: Type the email address associated with your account
3. **Check Your Email**: You'll receive a password reset link
4. **Reset Your Password**: Click the link and create a new password
5. **Login**: Use your new password to access the system

## Account Registration

New employees are typically registered by system administrators. If you need an account:
- Contact your HR department or system administrator
- Provide your employment details
- Wait for account creation confirmation
- Check your email for login credentials

## Security Features

The login system includes several security measures:

- **Password Encryption**: Passwords are securely encrypted
- **Session Management**: Automatic logout after inactivity
- **Failed Login Protection**: Account lockout after multiple failed attempts
- **Two-Factor Authentication**: Optional additional security layer
- **Email Verification**: Ensures account ownership

## Troubleshooting

### Common Login Issues

**"Invalid Credentials" Error**
- Double-check your email and password
- Ensure Caps Lock is not on
- Try the password recovery process

**"Account Locked" Message**
- Too many failed login attempts
- Wait 15-30 minutes and try again
- Contact your administrator if the issue persists

**Two-Factor Code Not Working**
- Ensure your device time is synchronized
- Try generating a new code
- Contact support if problems continue

**Email Verification Required**
- Check your email for a verification link
- Click the link to verify your account
- Contact support if you didn't receive the email

## Best Practices

- **Keep credentials confidential**: Never share your password
- **Use strong passwords**: Combine letters, numbers, and special characters
- **Log out when finished**: Especially on shared computers
- **Enable two-factor authentication**: For enhanced security
- **Regular password updates**: Change your password periodically
