# Events

## Overview

The Events feature allows administrators to create, manage, and track company events, meetings, trainings, and other organizational activities. Employees can view events, RSVP, and participate in scheduled activities.

## Purpose

The Events system is designed to:
- Organize company-wide and team events
- Manage event invitations and RSVPs
- Track event attendance
- Communicate event details to participants
- Coordinate meetings and trainings
- Maintain event history and records
- Facilitate team building and collaboration

## What Users Can See

### For Employees

#### Event Dashboard
- **Upcoming Events**: Events you're invited to
- **Event Calendar**: Visual timeline of events
- **My Events**: Events you've RSVPed to
- **Past Events**: Historical event records
- **Event Invitations**: Pending invites

#### Event Information
When viewing an event:
- **Event Title**: Name of the event
- **Description**: Detailed information
- **Date & Time**: When the event occurs
- **Duration**: Length of event
- **Location**: Where it takes place (physical or virtual)
- **Organizer**: Who created the event
- **Attendees**: List of invited participants
- **RSVP Status**: Your response (Going/Not Going/Maybe)
- **Attachments**: Related documents or files
- **Event Type**: Category (Meeting, Training, Social, etc.)

### For Administrators

#### Admin Event Dashboard
- **All Events**: Complete event list
- **Create Event**: Button to schedule new events
- **Event Management**: Edit, cancel, or duplicate events
- **Attendee Management**: Track responses and attendance
- **Event Reports**: Statistics and attendance records
- **Event Templates**: Reusable event formats

#### Admin Features
- Create and manage all events
- Invite specific employees or groups
- Track RSVPs and attendance
- Send event reminders
- Cancel or reschedule events
- View event analytics
- Export attendee lists

## How to Use Events

### For Employees: Participating in Events

#### Viewing Event Invitations

1. **Access Events**
   - Login to your dashboard
   - Click **"Events"** from main menu
   - See list of upcoming events

2. **View Event Details**
   - Click on any event
   - Event page opens with full information
   - Review date, time, location, description
   - See who else is invited
   - Check any attachments or materials

#### Responding to Event Invitations (RSVP)

1. **Open Event Details**
2. **Choose Your Response**:
   - Click **"Going"**: You will attend
   - Click **"Not Going"**: You cannot attend
   - Click **"Maybe"**: Unsure of attendance
3. **Add Comment** (optional):
   - Explain response if needed
   - Ask questions about the event
4. **Submit Response**
5. **Confirmation** message appears
6. Organizer is notified of your response

#### Updating Your Response

1. Go to **"My Events"**
2. Find the event
3. Click **"Change Response"**
4. Select new response
5. Update comment if needed
6. **Save** changes
7. Organizer notified of update

#### Viewing Event Calendar

1. Navigate to **"Events"** → **"Calendar"**
2. See events in calendar format
3. Color-coded by event type
4. Click on event for details
5. Filter by:
   - My events only
   - All events
   - Event type
   - Department

#### Event Reminders

Receive notifications before events:
- **Email reminders**: 1 day and 1 hour before
- **Dashboard notifications**: On event day
- **Calendar integration**: Synced to personal calendar

### For Administrators: Creating and Managing Events

#### Creating a New Event

1. **Access Event Creation**
   - Login to Admin Dashboard
   - Click **"Events"** from menu
   - Click **"Create Event"** button

2. **Fill Event Form**

   **Basic Information:**
   - **Event Title**: Clear, descriptive name
   - **Event Type**: Select category
     - Meeting
     - Training
     - Team Building
     - Company Event
     - Social Gathering
     - Conference
     - Workshop
     - Other
   - **Description**: Detailed event information
     - Purpose of the event
     - Agenda or schedule
     - What to bring/prepare
     - Any prerequisites

   **Date & Time:**
   - **Start Date**: When event begins
   - **Start Time**: Event start time
   - **End Date**: When event ends (for multi-day)
   - **End Time**: Event conclusion time
   - **All Day Event**: Check if applicable
   - **Time Zone**: Verify correct zone

   **Location:**
   - **Venue Type**: Physical or Virtual
   - **Physical Location**: 
     - Address
     - Room/building number
     - Directions or parking info
   - **Virtual Meeting**:
     - Meeting link (Zoom, Teams, etc.)
     - Meeting ID and password
     - Dial-in numbers

   **Attendees:**
   - **Invite All Employees**: Checkbox for company-wide
   - **Select Departments**: Choose specific teams
   - **Select Individuals**: Pick specific employees
   - **External Attendees**: Add external emails
   - **Maximum Capacity**: Limit attendees (optional)

   **Additional Settings:**
   - **RSVP Required**: Require response
   - **RSVP Deadline**: Last date to respond
   - **Allow Guests**: Permit +1 attendees
   - **Public Event**: Visible to all
   - **Reminder Settings**: When to send reminders
   - **Recurring Event**: Repeat pattern if applicable

   **Attachments:**
   - Upload relevant documents
   - Add presentation slides
   - Include agenda files
   - Attach reference materials

3. **Review Event Details**
   - Check all information is accurate
   - Verify attendee list
   - Confirm date and time
   - Review location details

4. **Save and Send Invitations**
   - Click **"Create Event"** or **"Send Invitations"**
   - Event is created
   - Invitations sent to all attendees
   - Event appears on calendar
   - Confirmation message displayed

#### Editing an Event

1. **Find the Event**
   - Go to **"Events"** → **"All Events"**
   - Locate event to modify
   - Click on event name

2. **Edit Event**
   - Click **"Edit Event"** button
   - Modify any field:
     - Update title or description
     - Change date/time
     - Modify location
     - Add/remove attendees
     - Update settings
   - **Save Changes**

3. **Notify Attendees**
   - Option to **"Notify attendees of changes"**
   - Check box if significant changes
   - Click **"Update"**
   - Change notifications sent automatically

#### Cancelling an Event

1. **Open Event Details**
2. Click **"Cancel Event"** button
3. **Provide Reason** for cancellation
4. Confirm cancellation
5. **Cancellation notices** sent to all attendees
6. Event marked as cancelled (not deleted)
7. Removed from active calendar

#### Managing Event Attendees

**View Attendees:**
1. Open event details
2. Click **"Attendees"** tab
3. See list with RSVP status:
   - ✓ Going (confirmed)
   - ✗ Not Going (declined)
   - ? Maybe (tentative)
   - ⊙ No Response (pending)

**Add Attendees:**
1. Click **"Add Attendees"**
2. Select employees
3. Click **"Send Invitations"**
4. New invitees notified

**Remove Attendees:**
1. Select attendee to remove
2. Click **"Remove"**
3. Confirm removal
4. Removed attendee notified

**Track Attendance:**
1. During/after event
2. Go to **"Attendees"** → **"Mark Attendance"**
3. Check off who actually attended
4. **Save** attendance record
5. Data recorded for reports

#### Event Reminders

**Automatic Reminders:**
- Default reminders sent:
  - 1 day before event
  - 1 hour before event
  - 10 minutes before (for meetings)

**Custom Reminders:**
1. Edit event
2. Go to **"Reminder Settings"**
3. Add custom reminder times
4. Select notification method (email, SMS, push)
5. **Save** settings
6. Reminders scheduled automatically

**Manual Reminder:**
1. Open event
2. Click **"Send Reminder Now"**
3. Select recipients:
   - All attendees
   - Only those who RSVPed
   - Only pending responses
4. Add custom message
5. Click **"Send"**

#### Event Reports and Analytics

1. **Access Reports**
   - Navigate to **"Events"** → **"Reports"**

2. **Available Reports:**
   - **Event Summary**: All events in period
   - **Attendance Report**: Who attended what
   - **RSVP Statistics**: Response rates
   - **Popular Events**: Most attended events
   - **Department Participation**: Engagement by team
   - **No-show Report**: RSVPed but didn't attend

3. **Generate Report:**
   - Select report type
   - Set date range
   - Choose filters (department, event type)
   - Click **"Generate"**
   - View or **Export** (PDF/Excel)

## Event Features

### Event Types and Categories

Organize events by type:
- **Meetings**: Team meetings, one-on-ones
- **Training**: Workshops, courses, certifications
- **Team Building**: Social events, activities
- **Company Events**: All-hands, town halls
- **Conferences**: External or internal conferences
- **Celebrations**: Birthdays, anniversaries, achievements
- **Volunteer**: CSR activities
- **Other**: Miscellaneous events

### Recurring Events

Create repeating events:
1. **Enable** "Recurring Event" when creating
2. **Select Pattern**:
   - Daily
   - Weekly (select days)
   - Monthly (by date or day of week)
   - Yearly
3. **Set Range**:
   - End by date
   - End after number of occurrences
   - No end date
4. **Save**
5. All instances created automatically
6. Edit single instance or entire series

### Event Capacity Management

For limited-capacity events:
1. Set **"Maximum Capacity"** when creating event
2. System tracks RSVPs
3. **Waitlist** activated when full
4. If someone declines, next person invited
5. Capacity status visible to admin

### Virtual Meeting Integration

If integrated with meeting platforms:
1. Select **"Virtual Event"**
2. Click **"Generate Meeting Link"**
3. System creates meeting:
   - Zoom, Teams, or configured platform
   - Meeting link auto-populated
   - Sent to attendees with invitation
4. **Join Meeting** button appears on event day

### Event Check-in

For physical events with check-in:
1. Enable **"Check-in Required"**
2. On event day, check-in screen available
3. Attendees check in:
   - QR code scan
   - Manual check-in by admin
   - Self-service check-in
4. Real-time attendance tracking
5. No-show reports generated

## Best Practices

### For Employees

- **Respond promptly**: RSVP as soon as possible
- **Update if plans change**: Change response if unable to attend
- **Check event details**: Review location, time, materials needed
- **Add to calendar**: Sync with personal calendar
- **Arrive on time**: Respect scheduled start time
- **Prepare**: Review attachments beforehand
- **Participate actively**: Engage during the event
- **Provide feedback**: Share thoughts after event

### For Administrators

- **Clear descriptions**: Provide comprehensive event information
- **Advance notice**: Create events well ahead of time
- **Choose right time**: Consider work schedules and time zones
- **Target invitations**: Invite relevant people only
- **Provide materials**: Share agenda and documents early
- **Send reminders**: Automated reminders for better attendance
- **Track attendance**: Record actual attendance
- **Follow up**: Share outcomes or minutes after
- **Review feedback**: Learn from each event
- **Archive properly**: Maintain event records

## Troubleshooting

### Common Issues

**Didn't Receive Event Invitation**
- Check spam/junk folder
- Verify email address in profile
- Ask organizer to resend
- Check event attendee list

**Cannot RSVP**
- RSVP deadline may have passed
- Event may be at capacity
- Check if you're on invite list
- Contact event organizer

**Event Not in Calendar**
- Only events you RSVPed "Going" to appear
- Refresh calendar
- Check calendar filters
- Verify calendar sync settings

**Meeting Link Not Working**
- Copy link and paste in browser
- Check meeting hasn't been rescheduled
- Verify meeting time hasn't passed
- Contact organizer for new link

**Event Reminder Not Received**
- Check notification settings
- Verify email preferences
- Ensure reminders are enabled for event
- Check spam folder

## Tips for Successful Events

- **Planning**: Start planning early
- **Communication**: Keep attendees informed
- **Flexibility**: Have backup plans (e.g., virtual option)
- **Engagement**: Make events interactive
- **Timing**: Choose convenient times
- **Follow-through**: Send thank you notes and summaries
- **Continuous improvement**: Learn from each event
- **Documentation**: Keep records and photos
- **Inclusivity**: Make events accessible to all
- **Fun**: Balance work events with social activities
