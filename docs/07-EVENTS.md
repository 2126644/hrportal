# Events

## Overview

The Events feature allows administrators to create and manage company events. Employees can view events they're invited to, see event details, and confirm or decline attendance.

## Purpose

The Events system is designed to:
- Organize company-wide and team events
- Manage event invitations
- Track event attendance (RSVP)
- Communicate event details to participants
- Coordinate meetings and activities

## What Users Can See

### For Employees

#### Event Dashboard
- **Upcoming Events**: Events you're invited to
- **Event Details**: Information about each event
- **Your RSVP Status**: Whether you've confirmed attendance

#### Event Information
When viewing an event:
- **Event Name**: Title of the event
- **Description**: Detailed event information
- **Date & Time**: When the event occurs
- **Location**: Where it takes place
- **Event Category**: Type of event (if categorized)
- **Event Status**: Upcoming, Ongoing, Completed, or Cancelled
- **Organizer**: Who created the event
- **Attendees**: List of invited participants
- **Your RSVP**: Your attendance confirmation status

### For Administrators

#### Admin Event Dashboard
- **All Events**: Complete event list
- **Create Event**: Button to schedule new events
- **Event Management**: Edit or delete events
- **Attendee Management**: View who is attending
- **Event Categories**: Organize events by type

#### Admin Features
- Create and manage all events
- Invite specific employees or departments
- Track RSVPs and attendance
- Edit or cancel events
- View attendee lists

## How to Use Events

### For Employees: Participating in Events

#### Viewing Event Invitations

1. **Access Events**
   - Login to your dashboard
   - Click **"Events"** from main menu
   - See list of events you're invited to

2. **View Event Details**
   - Click on any event
   - Event page opens with full information:
     - Event name and description
     - Date, time, and location
     - Who else is invited
     - Event category
   - Review all details

#### Confirming Attendance (RSVP)

1. **Open Event Details**
2. **Respond to Invitation**:
   - Click **"Confirm Attendance"** or **"Going"** - You will attend
   - Click **"Decline"** - You cannot attend
3. **Confirmation** message appears
4. Organizer can see your response
5. Your RSVP status is updated

#### Updating Your Response

1. Go to **"Events"**
2. Find the event
3. Click to open event details
4. **Change Response** by clicking different RSVP option
5. Updated response is saved
6. Organizer notified of change

#### Viewing Event Calendar

1. Navigate to **"Calendar"** (see [Calendar](06-CALENDAR.md))
2. Events you're invited to appear on your calendar
3. Click on event in calendar for details

### For Administrators: Creating and Managing Events

#### Creating a New Event

1. **Access Event Creation**
   - Login to Admin Dashboard
   - Click **"Events"** from menu
   - Click **"Create Event"** button

2. **Fill Event Form**

   **Basic Information:**
   - **Event Name**: Clear, descriptive title
   - **Description**: Detailed event information
     - Purpose of the event
     - Agenda or schedule
     - What to bring/prepare
     - Any special instructions

   **Date & Time:**
   - **Event Date**: When event takes place
   - **Event Time**: Start time of event

   **Location:**
   - **Event Location**: Where event takes place
     - Physical address/room number
     - Or "Virtual" if online
     - Include any access details

   **Category:**
   - **Event Category**: Select type (if categories exist)
     - Meeting
     - Training
     - Social
     - Company Event
     - Other

   **Attendees:**
   - **Invite Employees**: Select specific individuals
   - **Invite Departments**: Select entire departments
   - **Invite All**: Option for company-wide events (if available)

   **Additional:**
   - **Event Image**: Upload event banner/photo (optional)
   - **Tags**: Keywords for categorization (optional)

3. **Review Event Details**
   - Check all information is accurate
   - Verify attendee list
   - Confirm date and time
   - Review location details

4. **Save and Send Invitations**
   - Click **"Create Event"**
   - Event is created
   - Invitations sent to all attendees
   - Event appears on calendar

#### Editing an Event

1. **Find the Event**
   - Go to **"Events"** → **"All Events"**
   - Locate event to modify
   - Click **"Edit"** button or event name

2. **Update Information**
   - Modify any field:
     - Event name or description
     - Date/time
     - Location
     - Category
     - Add/remove attendees
   - **Save Changes**

3. **Notification**
   - Updated event information saved
   - Attendees notified of changes (if significant)

#### Cancelling an Event

1. **Open Event**
2. Update **Status** to **"Cancelled"**
   - Or use **"Delete"** option if available
3. **Save** changes
4. Attendees notified of cancellation

#### Managing Event Attendees

**View Attendees:**
1. Open event details
2. See **"Attendees"** section
3. View list with RSVP status:
   - ✓ Confirmed (accepted)
   - ✗ Declined
   - ⊙ No Response (pending)

**Add Attendees:**
1. Edit the event
2. Add new employees or departments
3. Save changes
4. New invitees notified

**View Attendance:**
1. After event occurs
2. Check who confirmed attendance
3. Compare with actual attendance (manual tracking)

## Event Features

### Event Categories

Organize events by type:
- Meetings
- Training sessions
- Social events
- Company events
- Team building
- Conferences
- Other categories (as configured)

### Event Status

Track event lifecycle:
- **Upcoming**: Event scheduled for future
- **Ongoing**: Event currently happening
- **Completed**: Event finished
- **Cancelled**: Event no longer happening

### RSVP Tracking

- Employees can confirm or decline
- See who is attending
- Track response rates
- No response shows as pending

### Calendar Integration

- Events appear on calendar
- Visual timeline of events
- Click calendar event to see details
- Helps with scheduling

### Department Invitations

- Invite entire departments at once
- All department members receive invitation
- Efficient for team or division events

## Best Practices

### For Employees

- **Respond promptly**: RSVP as soon as possible
- **Update if plans change**: Change response if unable to attend
- **Check event details**: Review location, time, and what to bring
- **Add to personal calendar**: Don't forget the event
- **Arrive on time**: Respect scheduled start time

### For Administrators

- **Clear descriptions**: Provide comprehensive event information
- **Advance notice**: Create events well ahead of time
- **Choose appropriate time**: Consider work schedules
- **Target invitations**: Invite relevant people
- **Provide materials**: Share agenda and documents
- **Track responses**: Monitor RSVPs
- **Follow up**: Send reminders if needed
- **Update promptly**: Notify of any changes immediately

## Troubleshooting

### Common Issues

**Didn't Receive Event Invitation**
- Check if you're on the invite list
- Look in Events section of portal
- Check email spam/junk folder
- Ask organizer to resend

**Cannot RSVP**
- Verify you're logged in
- Check event hasn't been cancelled
- Try different browser
- Contact event organizer

**Event Not in Calendar**
- Refresh calendar view
- Check calendar filters
- Verify you confirmed attendance
- Check date range being viewed

**Wrong Event Details**
- Contact event organizer
- Request correction
- Check for event updates

## Tips for Successful Events

### Planning
- **Start early**: Plan events in advance
- **Be specific**: Clear event details
- **Choose convenient times**: Consider attendee schedules
- **Confirm location**: Ensure space is available

### Communication
- **Detailed descriptions**: Explain event purpose
- **Clear logistics**: Time, place, what to bring
- **Reminders**: Send reminder before event
- **Updates**: Notify of any changes

### Follow-up
- **Track RSVPs**: Monitor who is coming
- **Send reminders**: Day before or morning of event
- **Update status**: Mark event as completed after
- **Gather feedback**: Learn for future events (if applicable)
