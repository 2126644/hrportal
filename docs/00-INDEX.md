## About the HR Portal

The HR Portal is a comprehensive human resources management system built with Laravel. It provides tools for:

- Employee management and profiles
- Attendance tracking and reporting
- Leave request and approval workflows
- Event and calendar management
- Task and project coordination
- Request submission and approval
- Form management (work handover, etc.)
- Announcements and communications

## User Roles

The system supports different user roles with varying levels of access:

- **Admin**: Full system access, can manage all employees, settings, and approvals
- **Employee**: Access to personal dashboard, can submit requests, view schedules
- **Approver**: Can approve requests and forms assigned to them

## Getting Support

If you need assistance or encounter any issues while using the HR Portal, please contact your system administrator.
